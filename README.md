# Just boat jump, nothing more

A simple Minecraft Fabric mod that adds the ability for boats to jump using the spacebar. Nothing more, nothing less!

## Features

- **Boat Jumping**: Press spacebar while in a boat to make it jump
- **Simple and Lightweight**: No configuration needed, just install and play
- **Works in Water and on Land**: Jump your boat wherever you are

## Usage

1. Get in a boat
2. Press **SPACEBAR** to jump
3. Enjoy!

## Requirements

- Fabric Loader 0.17.2+
